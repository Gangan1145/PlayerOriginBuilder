# PlayerOriginBuilder 玩家原点建造插件

- 作者: 淦
- 以玩家为原点放置物块，支持偏移、样式选择和多名玩家批量操作。

## 指令

| 语法 | 权限 | 说明 |
|------|------|------|
| `/pp <玩家名称> <X偏移> <Y偏移> <物块ID> [物块样式]` | `playerbuilder.use` | 在指定玩家周围放置物块|
| `/listplayers` 或 `/lp` | `playerbuilder.use` | 查看当前在线玩家及其坐标位置。|

### 参数说明
- **玩家名称**：支持精确匹配或部分匹配，使用 `all` 代表所有在线玩家。
- **X偏移**：水平偏移（正右负左，单位：格）
- **Y偏移**：垂直偏移（正下负上，单位：格）
- **物块ID**：要放置的物块数字 ID（可用 `/item list` 查询）
- **物块样式**：可选参数，用于火把、箱子等有多样式的物块（默认为 0）

### 使用示例

|指令                     |   作用            |
|----------------------|---------------------|
|` /pp 张三 5 -3 1 `      |    在张三右侧5格、上方3格放置石块 |
|` /pp all 0 0 30 `      |    在所有玩家脚下放置普通火把 |
|` /pp admin 10 0 169 `  |    在管理员右侧10格放置蓝玉块 |
|` /pp 李四 -3 2 21 `    |     在李四左侧3格、下方2格放置箱子 |
|` /pp VIP 0 -5 30 1 `  |     在名字含"VIP"的玩家上方5格放置红色火把 |

## 配置

> 本插件无需配置文件，所有功能通过命令实时控制。

- 若需要自定义物块样式映射或限制可放置物块
- 可手动修改源码中的 `SetTileFrame` 和 `GetTileName` 方法。
